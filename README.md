# covid-tracker

A website through which a country's covid cases and deaths can be viewed.
API used from https://covid19api.com/

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
